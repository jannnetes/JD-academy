JD Learn — mascot dachshund photos
===================================

Drop 4 transparent-background PNGs here with these EXACT names:

  dog-peek.png        — dachshund peeking from below (paws on edge), looking up
  dog-run.png         — dachshund side view (running pose)
  dog-headphones.png  — dachshund with headphones, looking up
  dog-lick.png        — dachshund sticking tongue out (click easter egg)

How it works:
- If /dog/dog-peek.png exists, the site automatically shows the PHOTO dachshund
  companion (peek by default, headphones near the courses section, lick on click,
  runs along the bottom as you scroll, speech bubbles).
- If the files are missing, the site falls back to the built-in 3D procedural dog.
  No broken images either way.

How to get transparent PNGs:
1. Take any dachshund photo (e.g. the reference shots).
2. Remove the background at https://remove.bg (free, up to a few images),
   or use Adobe Stock / Shutterstock "dachshund transparent png".
3. Export as PNG (transparent), name it as above, put it in this folder.
4. Refresh the site — the photo dog appears automatically.

Recommended size: ~600–900px tall, centered subject, trimmed transparent margins.
